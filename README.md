### Overview

This Project contains two `RTL8211F-CG` chips which is used to explore and demonstrate FPGA Ethernet application. This Project is compatible with [CUBE](https://github.com/AnterCreeper/CUBE) FPGA Devkit.  
该项目包含两个 `RTL8211F-CG` 芯片, 用于探索和展示基于 FPGA 的以太网应用。该项目与 [CUBE](https://github.com/AnterCreeper/CUBE) 开发工具搭配。
